# WDP 基础属性子技能

只负责基础属性域，不负责相机、覆盖物和前端布局。

## 前置条件

1. `App` 已初始化且渲染可用。
2. 依赖场景状态的操作在 `SceneReady(100%)` 后执行。
3. 输入参数完成必填、类型、范围校验。

## 标准流程

1. 明确属性目标。
- 定义读取或更新的属性项与目标值。

2. 执行读取或更新。
- 优先先读后改，避免盲写。

3. 回读确认结果。
- 更新后再次读取并核对一致性。

4. 处理失败分支。
- 记录属性名、输入摘要、错误对象。

## 质量门槛

1. 不在未满足时序时调用属性 API。
2. 不跳过参数校验直接写关键属性。
3. 不省略更新后的回读确认。

## 高频问题

1. 更新无效。
- 检查调用时机、参数类型、目标对象状态。

2. 更新后表现异常。
- 检查是否被后续调用覆盖。

3. 偶发失败。
- 检查并发冲突与重试节奏。

始终输出：
1. 属性调用链路改动点
2. 对行为一致性的影响
3. 验证步骤与结果
4. 最小复现与修复说明
