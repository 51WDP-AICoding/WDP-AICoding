## 使用约束

1. 不在仓库保存后台 token。

3. 若平台地址变更，只更新本文档并同步相关 skill 引用。
4. 第一轮编写/补完默认只参考第一优先在线文档；其他资料仅在用户明确要求交叉验证时使用。
