# 系统意图与架构设计报告

## 原始诉求

在现有WDP前端工程基础上，新增**纵向拆楼功能按钮**：
- 使用"实体移动"方法实现
- 将整个建筑（独立实体）垂直抬升**200米**
- 移动过程控制在**1.5秒内**完成
- 再次点击按钮则**下降复位**（toggle行为）

## 子任务拆解

| 序号 | 子任务 | 说明 |
|:---:|:---|:---|
| 1 | UI按钮添加 | 在现有UI面板添加"拆楼演示"按钮 |
| 2 | 获取建筑实体 | 通过eid获取Hierarchy类型建筑实体 |
| 3 | 创建隐藏路径 | 创建垂直方向的Path作为移动轨迹（bVisible: false） |
| 4 | 创建Bound移动 | 使用App.Bound绑定建筑和路径，设置1.5秒移动时间 |
| 5 | 状态管理 | 维护拆楼状态（抬起/复位），防止重复触发 |
| 6 | 清理链路 | 页面卸载时停止移动、删除路径和Bound对象 |

## API 能力与调用顺序

### 核心API识别

根据需求分析，使用 **实体移动（App.Bound）** 实现过程动画：

```javascript
// 1. 通过eid获取建筑实体
const res = await App.Scene.GetByEids(["827500951593025536"]);
const buildingEntity = res.result[0];

// 2. 创建垂直路径（隐藏）
const path = new App.Path({
  polyline: {
    coordinates: [
      [lng, lat, originalZ],      // 起点
      [lng, lat, originalZ + 200] // 终点（抬升200米）
    ]
  },
  pathStyle: { type: "solid", width: 1, color: "00000000" },
  bVisible: false // 隐藏路径
});
await App.Scene.Add(path);

// 3. 创建Bound移动对象
const bound = new App.Bound({
  moving: buildingEntity,  // 移动对象：建筑实体
  path: path,              // 移动路径
  boundStyle: {
    time: 1.5,             // 1.5秒完成
    bLoop: false,          // 不循环
    state: "play"          // 开始移动
  }
});
await App.Scene.Add(bound);
```

## 技术实现要点

### 1. 获取建筑实体

```javascript
// 通过eid直接获取建筑实体
const res = await App.Scene.GetByEids(["827500951593025536"]);
const buildingEntity = res.result[0];

// 获取建筑当前位置
const entityInfo = await buildingEntity.Get();
const [lng, lat, z] = entityInfo.result.location;
```

### 2. 创建垂直路径（隐藏）

```javascript
// 创建垂直抬升路径（起点 -> 终点）
const createVerticalPath = (startZ, endZ) => {
  return new App.Path({
    polyline: {
      coordinates: [
        [lng, lat, startZ],  // 起点
        [lng, lat, endZ]     // 终点
      ]
    },
    pathStyle: {
      type: "solid",
      width: 1,
      color: "00000000",    // 透明色
      passColor: "00000000"
    },
    bVisible: false,         // 隐藏路径
    customId: `path-${Date.now()}`
  });
};
```

### 3. 创建Bound移动

```javascript
// 创建Bound移动对象
const createBoundMove = (entity, path, duration) => {
  return new App.Bound({
    moving: entity,          // 移动对象
    path: path,              // 移动路径
    boundStyle: {
      time: duration,        // 移动时间（秒）
      bLoop: false,          // 不循环
      bReverse: false,       // 正向移动
      state: "play"          // 开始移动
    },
    customId: `bound-${Date.now()}`
  });
};
```

### 4. Toggle逻辑

```javascript
let isBuildingRaised = false;  // 状态标记
let originalZ = null;          // 原始高度
let currentBound = null;       // 当前Bound对象
let currentPath = null;        // 当前Path对象

async function toggleBuildingLift() {
  // 清理之前的移动对象
  if (currentBound) {
    await currentBound.Delete();
    currentBound = null;
  }
  if (currentPath) {
    await currentPath.Delete();
    currentPath = null;
  }
  
  if (isBuildingRaised) {
    // 下降复位
    const path = createVerticalPath(originalZ + 200, originalZ);
    await App.Scene.Add(path);
    currentPath = path;
    
    const bound = createBoundMove(buildingEntity, path, 1.5);
    await App.Scene.Add(bound);
    currentBound = bound;
    
    isBuildingRaised = false;
  } else {
    // 首次抬升，保存原始高度
    if (originalZ === null) {
      const info = await buildingEntity.Get();
      originalZ = info.result.location[2];
    }
    
    // 抬升
    const path = createVerticalPath(originalZ, originalZ + 200);
    await App.Scene.Add(path);
    currentPath = path;
    
    const bound = createBoundMove(buildingEntity, path, 1.5);
    await App.Scene.Add(bound);
    currentBound = bound;
    
    isBuildingRaised = true;
  }
}
```

## 风险提示

1. **Bound移动限制**：`App.Bound`的`moving`参数支持Hierarchy实体，但建筑实体会沿路径整体移动
2. **路径清理**：每次移动后需要及时删除Path和Bound对象，避免内存泄漏
3. **状态同步**：确保UI按钮状态与建筑实际位置一致
4. **坐标精度**：获取建筑位置时使用`Get()`方法，确保坐标准确

## 验证步骤

1. 点击"拆楼演示"按钮，建筑应在1.5秒内垂直抬升200米
2. 再次点击按钮，建筑应在1.5秒内下降回原始位置
3. 多次点击应能正常toggle
4. 页面刷新后状态重置
